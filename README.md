# DecisionGuide

A lightweight, logic-based assistant that helps analysts, auditors, and governance teams make clear, consistent decisions using structured decision flows.

DecisionGuide removes confusion during assessments by guiding users through predefined yes/no logic rather than relying on subjective judgement.  
Built with Python + Streamlit, it is fully open-source, easy to extend, and safe to run locally.

DecisionGuide provides:

- No data uploads and no privacy risks  
- Clean UI, easy navigation, and instant results

## üì¶ How to Run Locally

Clone the repo:

```

git clone https://github.com/Adeshola3/DecisionGuide
cd DecisionGuide

```

Install dependencies:

```

pip install -r requirements.txt

```

Run the app:

```

streamlit run app.py

```

Open the browser link shown in your terminal (usually http://localhost:8501).
